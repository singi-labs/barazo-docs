title: Rules

---

Your forum's rules are the standards you moderate against, and the ones members see and agree to. You manage them from **Rules** in the admin area.

## Writing rules

Each rule has a **title** (up to 200 characters) and a **description**. Add as many as you need, and use the up and down controls to order them. The order you set is the order members see.

## Versioning

Rules are versioned. Editing a rule creates a new version rather than overwriting the old one, so the history is preserved. Deleting a rule archives it instead of erasing it, which keeps any earlier references intact. This matters when a member agreed to a specific version of your rules: the record of what they agreed to still exists.
