Barazo is a forum first. If you've used one before, most of this will feel familiar. The difference is underneath, in who owns the data. This section covers the parts you'll use day to day.

Some features are set by whoever runs the forum. Reactions, for example, might be likes on one forum and up/down votes on another. Where that's the case, we say so.

## The essentials

- [Topics and replies](/docs/using/features/topics-and-replies): start a discussion, reply in a thread, format with Markdown
- [Reactions](/docs/using/features/reactions): likes, votes, or emoji, depending on the forum
- [Search](/docs/using/features/search): find topics, replies, and people
- [Notifications](/docs/using/features/notifications): keep up with replies and mentions
- [Profiles](/docs/using/features/profiles): what others see, and the work that shows up there
- [Staying safe](/docs/using/features/safety): block, mute, report, and filter mature content

## Set by each forum

A few things vary from one Barazo forum to the next, because each community configures its own space:

- Which reaction style is in use
- Whether new posts are reviewed before they appear
- What categories exist and how they're arranged
- Whether mature content is filtered by default
- The forum's rules and code of conduct

When a feature depends on the forum's settings, its page tells you what to look for.
