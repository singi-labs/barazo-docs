Search sits in the top bar of every Barazo forum. Type a keyword, a phrase, or a name and Barazo looks across topics, replies, and profiles.

## What you can find

- **Topics** by title and body
- **Replies**, so a good answer buried in a long thread still surfaces
- **People** by handle or display name

## How matching works

Every forum has **full-text search**. It matches the words in your query against the words in posts, and ranks the closest matches first. This works out of the box on every Barazo forum.

Some forums also enable **semantic search**. Instead of matching words exactly, it matches meaning, so a search for "reset my password" can surface a post titled "can't log back in" even with no shared words. Whether this is on depends on the forum, since it needs an AI model configured by whoever runs it. When it's available, results include close-in-meaning matches alongside exact ones.

## Tips

- Start broad, then add words to narrow. Search ranks by relevance, so extra words sharpen the top results.
- Searching a person's handle jumps you toward their profile and posts.
- If a forum feels like it's missing search, the admin may have turned it off. Browse by category instead.
