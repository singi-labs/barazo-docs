title: Reactions
description: React to posts with a like, an up/down vote, or emoji, depending on how the forum is set up.

A reaction is a quick response to a post without writing a reply. Barazo supports three reaction styles, and **each forum picks one**. Some forums even set a different style per category. So what you see under a post depends on where you are.

## The three styles

**Like** is a single toggle. Tap the heart to show you like a post, and tap again to undo. The count shows how many people did. This is the default for new forums.

**Vote** gives you up and down arrows, like Reddit. The net score (ups minus downs) sits between the arrows. Votes also feed how the forum sorts posts, so a well-voted reply can rise to the top.

**Emoji** uses a set of emoji chosen by the forum's admins. You can react with more than one, and each emoji keeps its own count. This is the most expressive style and the most playful.

A forum can also turn reactions off entirely, in which case no reaction control appears.

## How to react

Find the reaction control under a post and select it. On emoji forums, a picker opens with the available set. Your reaction is public. Others can see the counts, and on emoji and like forums they can see who reacted.

## Reactions and sorting

On vote forums, reactions do double duty: they signal approval and they influence order. A topic sorted by "top" surfaces the highest-scored posts. On like and emoji forums, popularity is measured too, so active discussions can bubble up regardless of the style in use.

## Why it varies

Reactions are a forum setting, not a fixed part of Barazo. A support community might want plain likes, a link-sharing board might want votes, and a casual community might want emoji. The admin decides, so the forum matches how its people actually talk.
